# LibraryManagementSystem

## Project Description
The Library Management System is a C++ console-based application developed using Object-Oriented Programming and File Handling concepts.  
This project helps manage library books efficiently by allowing users to add, search, issue, and return books.

---

## Features
- Add new books
- Display all books
- Search books by title
- Search books by author
- Issue books
- Return books
- File handling for data storage

---

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Code::Blocks IDE

---

## Functionalities

### Add Book
Allows the user to add new book details including:
- Book ID
- Book Title
- Author Name

### Display Books
Displays all available books stored in the file.

### Search By Title
Searches for books using the book title.

### Search By Author
Searches for books using the author name.

### Issue Book
Issues a selected book to the user.

### Return Book
Returns an issued book.

---

## File Used
- `library.txt` → Stores all book records permanently.

---

## How to Run
1. Open the project in Code::Blocks.
2. Open `main.cpp`.
3. Click **Build and Run** or press `F9`.
4. Use menu options to perform operations.
